

# JAINU-Model　(T5 fine-tuned model)
JAINU is a Japanese - Ainu language machine translation model.
We used sonoisa t5-base-japanese (https://huggingface.co/sonoisa/t5-base-japanese).


# Examples

| input | output|
|---|---|
|こんにちは！|イランカラプテ!|
|ありがとう|ヒオーイオイ|
|ありがとうございます|イヤイライケレ|
|熊は神ですか|キムンカムイアナクカムイネルウェ?|
|怖いものです|アシトマプネナ|
